# Pyreon Fundamentals

Ecosystem packages for the [Pyreon](https://github.com/pyreon/pyreon) framework — data fetching, forms, i18n, testing utilities, and starter templates.

## Packages

| Package | Description | Status |
| --- | --- | --- |
| `@pyreon/query` | TanStack Query adapter — `useQuery`, `useMutation`, Suspense, SSR dehydration | Moved from core |
